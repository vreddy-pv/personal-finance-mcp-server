## Scripts

- **Install MCP Server Dependencies**: `sh scripts/install-mcp.sh`
  - Installs the Python dependencies for the MCP server.

- **Start MCP Server**: `sh scripts/start-mcp.sh`
  - Starts the Uvicorn server for the MCP application.
